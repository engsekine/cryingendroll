# Next.js と microCMS で作る web サイト

##　デプロイ URL
https://sekinedaichi.com/

## .env ファイル設定

- `cp .env.sample .env`

```
API_KEY=
SERVICE_ID=
NEXT_PUBLIC_MAIL_USER=
NEXT_PUBLIC_MAIL_PASS=
NEXT_PUBLIC_TWITTER=
NEXT_PUBLIC_TWITTER_SITE=
NEXT_PUBLIC_FACEBOOK=
NEXT_PUBLIC_FACEBOOK_APPID=
NEXT_PUBLIC_INSTAGRAM=
NEXT_PUBLIC_URL=
```

## yarn ビルド

`yarn build`

## yarn 起動

`yarn dev:next`
